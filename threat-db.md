## Cross-Chain Memory

When a threat is detected on one chain, it's automatically flagged across all chains:

**Example:**
- Scammer wallet detected on Ethereum → flagged on Solana, Base, BSC
- Malicious contract pattern on Base → blocked across all EVMs
- Known drainer domain → blacklisted universally

One brain. Every chain. Nothing slips through.

## Threat Removal Process

If a platform is incorrectly flagged:

1. Submit evidence via GitHub issue at:  
   https://github.com/PhoebeBoss/phantom-browser-agent/issues

2. Include:
   - Proof of legitimacy (audit report, team doxx, track record)
   - Explanation of false positive trigger
   - Request for score recalculation

3. Review within 24 hours

4. If valid, threat removed and score updated
